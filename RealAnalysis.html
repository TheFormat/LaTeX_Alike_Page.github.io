Abstract

нқ . measurable functionмқ„ мһҳ лӘЁлҘҙкІ м–ҙм„ң кұ°кё°м„ңл¶Җн„° мӢңмһ‘н•ҳкІ лӢӨ.

мҡ°м„  мқҙ кіјлӘ©мқ„ мһҳн•ҳл Өл©ҙ н•ҙм„қн•ҷмқ„ м •л§җ мһҳн•ҙм•ј н•ңлӢӨлҚ”лқј. supкіј infм—җ лҢҖн•ҙм„ң м§ҡкі  л„ҳм–ҙк°Җмһҗ.
Supremum(the least upper bound)
Let $A\subset \mathbb{R}$. Then,

  1. $a\in [-\infty,\infty]$ is an upper bound of $A$ if $x\leq a$ $\forall x\in A$
  2. $a$ is a supremum of A if $a$ is an upper bound of $A$, and $a\leq b$ if $b$ is an upper bound of $A$.

мӨ‘мҡ”н•ң м„ұм§Ҳ н•ҳлӮҳ м§ҡкі  л„ҳм–ҙк°Җмһҗ. $a = \sup A, A \neq \emptyset$. Then, $$\forall \epsilon >0, \exists x\in A \text{ s.t } x\geq a-\epsilon$$ This is same to $$\exists x_{n}\in A \text{ s.t } x_{n}\uparrow a$$ $a$к°Җ $\sup$мқҙлқјлҠ” мҶҢлҰ¬лҠ” н•ӯмғҒ $a-\epsilon$ліҙлӢӨ нҒ¬кұ°лӮҳ к°ҷмқҖ A мӣҗмҶҢлҘј м°ҫмқ„ мҲҳ мһҲлӢӨлҠ” мҶҢлҰ¬мқҙмһҗ, $a$лЎң мҰқк°Җн•ҳлҠ” мҲҳм—ҙмқ„ мһЎмқ„ мҲҳ мһҲлӢӨлҠ” мҶҢлҰ¬мқҙлӢӨ.
мқҙм ң мҡ°лҰ¬лҠ” measure $\lambda$лҘј м •мқҳн•  кІғмқҙлӢӨ. $\lambda : A \mapsto \lambda(A)\in [0,\infty]$лҠ” set functionмқҙлӢӨ. м •лҰ¬н•ҙм„ң л§җн•ҳмһҗл©ҙ

Roughly speacking, a measure $\lambda$ on $X$ is a rule which assigns a number to subsets on $X$ s.t

  1. $\lambda(\emptyset) = 0$
  2. $\lambda(\bigcup\limits_{k=1}^{\infty}A_{k}) = \sum\limits_{k=1}^{\infty}\lambda(A_{k})$ if $A_{i}\cap A_{j} = \emptyset$ for $i\neq j$
countable additivity лҸ„ л§ҢмЎұн•ҳлҠ” set functionмқҙ лҗҳкІ лӢӨ.

Let $M$ be a collection of subsets of $X$. That is, $M\subset 2^{X}.$
$M$ is an algebra(field) if
a) $\emptyset\in M$
b) $A,B\in M\Rightarrow A\cup B\in M$
c) $A\in M\Rightarrow A^{c}\in M$
An algebra $M$ is a $\sigma$-algebra if
b') $A_{k}\in M\Rightarrow \bigcup\limits_{k=1}^{\infty}A_{k}\in M$
Let $X$ given, and $N$ be a collection of subsets of $X$.($N\subset 2^{X}$)
$\sigma(N) := \text{intersection of all $\sigma$-algebra containing $N$}$.
Here, it doesn't need to be countable intersection.

Know that
1. $\sigma(N)$ is a $\sigma$-algebra
2. $\sigma(N)$ is the smallest $\sigma$-algebra containing $N$.(generated by $N$)
мҰқлӘ…мқҖ л„ҳм–ҙк°ҖкІ лӢӨ.

$X = \mathbb{R}^{n}, N = \text{collection of all open sets}$
$\sigma(N) = \text{Borel $\sigma$-algebra} = \mathcal{B}(\mathbb{R}^{n})$
If $A\in \mathcal{B}(\mathbb{R}^{n})$, then $A$ is called a Borel set.

By using this, we can generalize it to..

$X$ is a topological space, $N$ is the collection of all open sets in $X$.
$\sigma(N) = \text{Borel $\sigma$-algebra} = \mathcal{B}(X)$
If $A\in \mathcal{B}(X)$, then $A$ is called a Borel set.

лӘЁл“  open set, closed set, countable unionл“ӨмқҖ Borel SetмқҙлӢӨ. мҰү, $\{1\}$к°ҷмқҖ м§‘н•©мқҖ $\bar{\mathbb{R}}$мқҳ Borel Setмқҙ лҗҳкІ лӢӨ.
мқҙм ң standard topologyм—җм„ңмқҳ $X = \mathbb{R}$м—җм„ң neighborhood $Br(X)$лҘј м°ҫм•„ліҙмһҗ.
$(a,b),(a,b],(a,\infty),(-\infty,a)$к°Җ л§Ңл“ңлҠ” $\sigma$-algebraк°Җ $\mathcal{B}(\mathbb{R})$мҷҖ к°ҷлӢӨ.
мҰқлӘ…мқҖ л„ҳм–ҙк°Җмһҗ. м°ёкі лЎң extended real number $\bar{\mathbb{R}}$м—җм„ң open setмқ„ мһҳ м •мқҳн•ҙм„ң н‘ңнҳ„н•ҳлҠ” кІғлҸ„ к°ҖлҠҘн•ҳлӢӨ.
нӣ„мҲ н•  н•ЁмҲҳ $f$м—җ лҢҖн•ҙм„ңлҠ” $\mathbb{R}$ лҢҖмӢ  $\bar{\mathbb{R}}$мңјлЎң м •мқҳн•ҳкІ лӢӨ.
м–јл§Ҳ м•ҲлӮЁм•ҳлӢӨ. measurable spaceлҘј м •мқҳн•ҳмһҗ.

A pair $(X,M)$ is called a measurable space if $M$ is a $\sigma$-algebra on $X$.
Let $(X,M)$ be a measurable space, and $Y$ be a topological space.
A function $f:X\rightarrow Y$ is called M-measurable if $\forall B \in \mathcal{B}(Y), f^{-1}(B)\in M$.

лӢӨмӢңл§җн•ҙ, Borel Setмқҳ inverse imageк°Җ $\sigma$-algebraл©ҙ мҡ°лҰ¬лҠ” $f$лҘј M-measureableмқҙлқјкі  л¶ҖлҘёлӢӨ.
м•„мү¬мҡҙ кІҢ мһҲлӢӨл©ҙ, лӘЁл“  Borel Setм—җ лҢҖн•ҙм„ң к·ёкұ°мқҳ inverse imageк°Җ $\sigma$-algebraм—җ л“Өм–ҙк°Җм•ј н•ңлӢӨлҠ” кІғмқҙлӢӨ.
к·ёлһҳм„ң, м•„лһҳ propositionмқ„ нҶөн•ҙм„ң measurable functionмқ„ нҷ•мқён•ҳлҠ”кІҢ нҺён•ҳлӢӨлҚ”лқј.

Suppose $N\subset \mathcal{B}(Y)$ s.t $\sigma(N) = \mathcal{B}(Y)$.
Then, $f:X\rightarrow Y$ is M-measurable iff $\forall B\in N, f^{-1}(B)\in M$.
$(X,M)$ : measurable space, $f:X\rightarrow Y$.
$$N=\{B\subset Y \mid f^{-1}(B)\in M\}$$ Then, $N$ is a $\sigma$-algebra on $Y$.

мҰқлӘ…мқҖ л„ҳм–ҙк°ҖлҸ„лЎқ н•ҳкІ лӢӨ. Prop 2лЎң Prop 1мқ„ мҰқлӘ…н•  мҲҳ мһҲлӢӨ.
мӨ‘мҡ”н•ң кұҙ, мҡ°лҰ¬лҠ” мқҙм ң лӘЁл“  Borel Setмқ„ мІҙнҒ¬н•ҳлҠ” лҢҖмӢ  $N$м—җ мһҲлҠ” кІғл“Өл§Ң мІҙнҒ¬н•ҳл©ҙ лҗңлӢӨлҠ” кІғмқҙлӢӨ.
мқҙлҘјн…Ңл©ҙ $\mathcal{B}(\mathbb{R})$м—җм„ң $N_{1} = \{\text{open sets}\}, N_{2} = \{(a,b)\mid a < b\}, N_{3} = \{[a,\infty)\mid a \in \mathbb{R}\}$лЎң мһЎмқ„ л•Ң
$\sigma(N_{1}) = \sigma(N_{2}) = \sigma(N_{3}) = \mathcal{B}(\mathbb{R})$мқҙлӢӨ.
Proposition 1м—җ мқҳн•ҙм„ң $\bar{\mathbb{R}}$м—җм„ңмқҳ н•ЁмҲҳ $f$к°Җ M-measurableн•ңм§Җ нҷ•мқён•  л°©лІ•мқҙ мғқкІјлӢӨ.

$f : X \rightarrow \bar{\mathbb{R}}$ is M-measurable if $$\{x \mid f(x)\leq a\} = f^{-1}([-\infty,a])\in M \text{ for all } a \in \mathbb{R}$$

м•„лһҳ Corollaryм—җ мқҳн•ҙм„ң continuous functionмқҖ measurable functionмқҙ лҗңлӢӨ.

If $Y = \mathbb{R}, M = \mathcal{B}(\mathbb{R})$ and $f:\mathbb{R}\rightarrow \mathbb{R}$ is continuous, then $f$ is measurable.

мҰқлӘ…мқҖ к°„лӢЁн•ҳлӢӨ. continuous functionмқҳ м •мқҳм—җ мқҳн•ҙ open setмқҳ inverse imageлҠ” open setмқҙ лҗңлӢӨ. к·ёлҹ¬лҜҖлЎң $f^{-1}(B)\in M$мқҙ лҗңлӢӨ.

лӢӨмӢңл§җн•ҙ, $f, g$к°Җ M-measurableн•ҳл©ҙ $|f|^{p}, \frac{1}{f}$(лӢЁ, $f\neq 0$), $f+g, fg, f\circ g$лҸ„ M-measurableн•ҳлӢӨ.
