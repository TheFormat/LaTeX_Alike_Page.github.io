Supremum(the least upper bound)
Let $A\subset \mathbb{R}$. Then,

  1. $a\in [-\infty,\infty]$ is an upper bound of $A$ if $x\leq a$ $\forall x\in A$
  2. $a$ is a supremum of A if $a$ is an upper bound of $A$, and $a\leq b$ if $b$ is an upper bound of $A$.

중요한 성질 하나 짚고 넘어가자. $a = \sup A, A \neq \emptyset$. Then, $$\forall \epsilon >0, \exists x\in A \text{ s.t } x\geq a-\epsilon$$ This is same to $$\exists x_{n}\in A \text{ s.t } x_{n}\uparrow a$$ $a$가 $\sup$이라는 소리는 항상 $a-\epsilon$보다 크거나 같은 A 원소를 찾을 수 있다는 소리이자, $a$로 증가하는 수열을 잡을 수 있다는 소리이다.
이제 우리는 measure $\lambda$를 정의할 것이다. $\lambda : A \mapsto \lambda(A)\in [0,\infty]$는 set function이다. 정리해서 말하자면

Roughly speaking, a measure $\lambda$ on $X$ is a rule which assigns a number to subsets on $X$ s.t

  1. $\lambda(\emptyset) = 0$
  2. $\lambda(\bigcup\limits_{k=1}^{\infty}A_{k}) = \sum\limits_{k=1}^{\infty}\lambda(A_{k})$ if $A_{i}\cap A_{j} = \emptyset$ for $i\neq j$
countable additivity 도 만족하는 set function이 되겠다.

Example 3에서는 우리가 배울 Lebesgue measure on $\mathbb{R}^{n}$를 확인할 수 있다. $$A = [a_{1},b_{1}]\times [a_{2},b_{2}]\times \cdots \times [a_{n},b_{n}](-\infty < a_{i} \leq b_{i} < \infty, i=1,2,\ldots,n)$$ $$\lambda(A) = \prod_{i=1}^{n}(b_{i}-a_{i}) = \text{volume of } A$$ 아쉽게도, 모든 subset에 대해서 measure를 정의하지는 않았다. 어떻게 arbitrary set에 대해서 $\lambda(A)$를 정의할까?
Construction이 필요하다.

A construction of Lebesgue measure on $\mathbb{R}^{n}$

Stage 0. $\lambda(\emptyset) = 0$
Stage 1. "special rectangle" $I = [a_{1},b_{1}]\times [a_{2},b_{2}]\times \cdots \times [a_{n},b_{n}]$ $$\lambda(I) \equiv \prod_{i=1}^{n}(b_{i}-a_{i}) = \text{volume of } I$$ Remark 하나를 보자. special rectange $I$가 non-overlapping special rectangles의 finite union이면 $\lambda(I) \equiv \sum_{i=1}^{n} Vol(I_i) = \sum_{i=1}^{n} \lambda(I_i)$임을 알 수 있다.
여기서 non-overlapping이라 함은 $I_{i}\cap I_{j} = \emptyset$ for $i\neq j$를 의미한다.
Stage 2. "special polygon" A set $P$ is a special polygon if it is a finite union of special rectangles of positive measure. That is, $P = \bigcup\limits_{i=1}^{n}I_{i}$ such that $\lambda(I_{i}) > 0$ for $i=1,2,...,n$. $$\lambda(P) \equiv \sum_{j=1}^{m} \lambda(I'_{j})$$ 갑자기 $I'$가 등장했다. 이는 $P$가 위의 Remark로 인해 non-overlapping special rectangle $I_{j}'$로 분해할 수 있기 때문이다.
넘어가기 전 fuction이 되는지를 체크해야 할 것이다. 다른 non-overlapping하는 $I'$들에 대해서 $\sum$이 같음을 보이면 된다. 혼자하자.
open set $(0,1) = \bigcup\limits_{n=2}^{\infty}[\frac{1}{n},1-\frac{1}{n}]$을 보자. open set들은 안에서부터 만들어나간다. 극한이 있는지 없는지는 모르니 measure을 $\sup$로 정의하자.
Stage 3. Let $G$ be a non-empty open set. Then, $$\lambda(G) = \sup\{\lambda(P) \mid P\subset G\}>0$$ Note that if $G$ is open, then $\lambda(G)$ = 0 iff $G = \emptyset$.
Stage 4. Let $K$ be a compact set. Then, $$\lambda(K) = \inf\{\lambda(G) \mid K\subset G \text{ where } G\text{ is open}\}$$ 이 정의해서 조심해야 할 점은 $K$가 special polygon일 때에는 measure가 두 가지로 정의된다는 점이다. 두 정의가 일치하는지 확인해야 한다는 점은 명심하자. 바로 성질들로 넘어가겠다.

Let $M$ be a collection of subsets of $X$. That is, $M\subset 2^{X}.$
$M$ is an algebra(field) if
a) $\emptyset\in M$
b) $A,B\in M\Rightarrow A\cup B\in M$
c) $A\in M\Rightarrow A^{c}\in M$
An algebra $M$ is a $\sigma$-algebra if
b') $A_{k}\in M\Rightarrow \bigcup\limits_{k=1}^{\infty}A_{k}\in M$
Let $X$ given, and $N$ be a collection of subsets of $X$.($N\subset 2^{X}$)
$\sigma(N) := \text{intersection of all $\sigma$-algebra containing $N$}$.
Here, it doesn't need to be countable intersection.

Know that
1. $\sigma(N)$ is a $\sigma$-algebra
2. $\sigma(N)$ is the smallest $\sigma$-algebra containing $N$.(generated by $N$)
증명은 넘어가겠다.

$X = \mathbb{R}^{n}, N = \text{collection of all open sets}$
$\sigma(N) = \text{Borel $\sigma$-algebra} = \mathcal{B}(\mathbb{R}^{n})$
If $A\in \mathcal{B}(\mathbb{R}^{n})$, then $A$ is called a Borel set.

By using this, we can generalize it to..

$X$ is a topological space, $N$ is the collection of all open sets in $X$.
$\sigma(N) = \text{Borel $\sigma$-algebra} = \mathcal{B}(X)$
If $A\in \mathcal{B}(X)$, then $A$ is called a Borel set.

모든 open set, closed set, countable union들은 Borel Set이다. 즉, $\{1\}$같은 집합은 $\bar{\mathbb{R}}$의 Borel Set이 되겠다.
이제 standard topology에서의 $X = \mathbb{R}$에서 neighborhood $Br(X)$를 찾아보자.
$(a,b),(a,b],(a,\infty),(-\infty,a)$가 만드는 $\sigma$-algebra가 $\mathcal{B}(\mathbb{R})$와 같다.
증명은 넘어가자. 참고로 extended real number $\bar{\mathbb{R}}$에서 open set을 잘 정의해서 표현하는 것도 가능하다.
후술할 함수 $f$에 대해서는 $\mathbb{R}$ 대신 $\bar{\mathbb{R}}$으로 정의하겠다.
얼마 안남았다. measurable space를 정의하자.

A pair $(X,M)$ is called a measurable space if $M$ is a $\sigma$-algebra on $X$.
Let $(X,M)$ be a measurable space, and $Y$ be a topological space.
A function $f:X\rightarrow Y$ is called M-measurable if $\forall B \in \mathcal{B}(Y), f^{-1}(B)\in M$.

다시말해, Borel Set의 inverse image가 $\sigma$-algebra면 우리는 $f$를 M-measureable이라고 부른다.
아쉬운 게 있다면, 모든 Borel Set에 대해서 그거의 inverse image가 $\sigma$-algebra에 들어가야 한다는 것이다.
그래서, 아래 proposition을 통해서 measurable function을 확인하는게 편하다더라.

Suppose $N\subset \mathcal{B}(Y)$ s.t $\sigma(N) = \mathcal{B}(Y)$.
Then, $f:X\rightarrow Y$ is M-measurable iff $\forall B\in N, f^{-1}(B)\in M$.
$(X,M)$ : measurable space, $f:X\rightarrow Y$.
$$N=\{B\subset Y \mid f^{-1}(B)\in M\}$$ Then, $N$ is a $\sigma$-algebra on $Y$.

증명은 넘어가도록 하겠다. Prop 2로 Prop 1을 증명할 수 있다.
중요한 건, 우리는 이제 모든 Borel Set을 체크하는 대신 $N$에 있는 것들만 체크하면 된다는 것이다.
이를테면 $\mathcal{B}(\mathbb{R})$에서 $N_{1} = \{\text{open sets}\}, N_{2} = \{(a,b)\mid a < b\}, N_{3} = \{[a,\infty)\mid a \in \mathbb{R}\}$로 잡을 때
$\sigma(N_{1}) = \sigma(N_{2}) = \sigma(N_{3}) = \mathcal{B}(\mathbb{R})$이다.
Proposition 1에 의해서 $\bar{\mathbb{R}}$에서의 함수 $f$가 M-measurable한지 확인할 방법이 생겼다.

$f : X \rightarrow \bar{\mathbb{R}}$ is M-measurable if $$\{x \mid f(x)\leq a\} = f^{-1}([-\infty,a])\in M \text{ for all } a \in \mathbb{R}$$

아래 Corollary에 의해서 continuous function은 measurable function이 된다.

If $Y = \mathbb{R}, M = \mathcal{B}(\mathbb{R})$ and $f:\mathbb{R}\rightarrow \mathbb{R}$ is continuous, then $f$ is measurable.

증명은 간단하다. continuous function의 정의에 의해 open set의 inverse image는 open set이 된다. 그러므로 $f^{-1}(B)\in M$이 된다.

이제 measurable function의 성질을 알아보자. measurable function의 합, 곱, composition이 measurable function이 된다.
다시말해, $f, g$가 M-measurable하면 $|f|^{p}, \frac{1}{f}$(단, $f\neq 0$), $f+g, fg, f\circ g$도 M-measurable하다.
나아가 measurable function의 limit도 measurable function이 된다. sup, inf, limsup, liminf 모두 measurable하다. 귀찮아서 수식은 안적겠다.
