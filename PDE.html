Linearity
Let $\mathcal{L}$ be an operator.(mapping a function to another function) Then, an operator $\mathcal{L}$ is linear if
$\mathcal{L}(u+v) = \mathcal{L}u + \mathcal{L}v. \mathcal{L}(cu) = c\mathcal{L}(u)$.
For example, from $u_{x}+yu_{y}-5x = 0$, $(\frac{\partial}{\partial x}+ y\frac{\partial}{\partial y})$ is a linear operator.
On the other hand, from $u_{x}+u\cdot u_{y}-5x = 0$, $\mathcal{L}u := u_{x}+u\cdot u_{y}$ is not a linear operator. Check it by yourself.
A PDE $\mathcal{L}u = f(x,y,...)$ where $f$ is independent of $u$ is linear if $\mathcal{L}$ is a linear operator.

A linear PDE $\mathcal{L}u = f(x,y,...)$ is homogeneous if $f(x,y,...) = 0$. Otherwise, we say the PDE is non-homogeneous.

(Principle of Superposition)
If $u_{1},u_{2}$ are solutions, then so is $c_{1}u_{1}+c_{2}u_{2}$ for any $c_{1}, c_{2}$.

Now, lets solve for $u=u(x,y)$ satisfying $au_{x}+bu_{y}= 0, a,b\in \mathbb{F}$, one of them is non-zero.
Recall that $\nabla u = \langle u_{x}, u_{y} \rangle$. Then, $\nabla u \cdot \langle a,b \rangle = 0$ indicates that $u$ is a constant in the direction $\langle a,b \rangle.$ $$\frac{dy}{dx} = \frac{b}{a} \text{ or } \frac{x-x_{0}}{a} = \frac{y-y_{0}}{b}, \text{ thus, we have } bx-ay = C$$ $u$ only depends on $C$. Thus, $u(x,y) = f(C) = f(bx-ay), f(ay-bx)$.
예시를 보자. $u_{x}+2xy^{2}u_{y} = 0$에서 $u$는 항상 $\langle 1, 2xy^{2} \rangle$에서 항상 상수일 것이다. 조금 더 정리하면
$u(x,y) = f(C) = f(x^{2}+\frac{1}{y})$이다.

다른 방법도 있다. Change of Coordinates의 경우인데, 위의 예시에서 같이 보자.
We know that the characteristic curve is $x^{2}+\frac{1}{y} = C$. Let $\xi = x^{2}+\frac{1}{y}, \eta = x$. Then, the Jacobian is $$J(\xi,\eta) = \begin{vmatrix} \frac{\partial \xi}{\partial x} & \frac{\partial \xi}{\partial y} \\ \frac{\partial \eta}{\partial x} & \frac{\partial \eta}{\partial y} \end{vmatrix} = \frac{1}{y^{2}} \neq 0$$ Thus, we can change the coordinates. From $u_{x} = 2xu_{\xi} + 1\cdot u_{\eta}, u_{y} = (-\frac{1}{y^{2}})u_{\xi}$, $u_{\eta} = 0$
$u = f(\xi) = f(x^{2}+\frac{1}{y})$.
